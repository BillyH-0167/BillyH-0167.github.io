Application Center Set Up

Information:

The application center is a very customizable and user friendly system which allows a player to do an application, and once passed, it will automatically give them a rank you specify!


Setup Step 1: Getting needed infomation

In order to configure the application center, you need the items listed below.

Items:

  • Valid License
  • Valid Key
  • Personal Order Channel in our discord
  • Fully configured bot (Click here to see how to configure the bot)
  • Download the Template: Click here to download the Template


Once you have the items above, continue reading.

Step 2: Configuration

Open the application center template you have downloaded.

What it should look like when you open it:

Now, publish the game and turn on HTTP Requests, third party teleports, and studio api (optional)


How to publish the game
To publish a game, head on over to the top left corner and hit file.
Then press "Publish to Roblox"


How to turn on required services
To turn on required services, make sure the game is published then head over to the game settings tab.
Now press "Security"
Then turn on allow http requests, third party teleports, and third party sales.


Once you've done that, head over to the Explorer tab and click the little arrow next to ServerScriptService.


Next, double click the module named Configuration and open it.


When you open the module, you should see this. Here is where all of the configuration for the App Center is done.


The setup for the this template is very basic, lets now talk about the variables and how you add/remove questions to the application.

Group Variable :

  • The group Variable is used to define what group this rank center is for. Remove the numbers that are in it and replace it with your group ID. (Click the Click to see arrow below)

Getting group ID + setting variable:

Click to see

  1. Head to your group on the ROBLOX page.

  2. Go to the top of your search bar and double click on the numbers.

  3. Then, right click and press copy.

  4. Next, paste that into the Group variable.

You're now done with that!


GameKey Variable :

  • The game Key variable is set the place ID. DO NOT CHANGE THIS. You will have to white list the place via bot as seen later.

Lives Variable :

  • This variable is basically the number of questions a user can get wrong before they fail. By default it is set to 3, but you're able to make it whatever you want.

PassRank Variable :

  • This variable is the rank the user will get once they pass the application.

Amount Variable :

  • This variable is the amount of question in the application. Be sure that it matches the amount of questions you have inside of the questions dictionry.

-- End of Variables--


Adding/Removing Questions:

Adding and removing questions is a very basic process. In order to do this, find the questions dictionary, it should look like this:

By default the application center comes with three premade questions, replace each value with whatever you want for it.


Values in the Questions dictionary :

  • Question number: Put in the question number.
  • Question: Put the question you want the user to answer iside of the quotes.
  • Answers: 4 Answer choices are required. Replace whatever is in the quotes with the answer you want to put.
  • Correct answer: put the number of the correct answer.

Adding questions:

To add a question, copy one of the question dictionaries, and paste it after an };. Then change the question number to the question it is.

Question1 = {
            Question = "Who is god?";
            Answer1 = "RGM";
            Answer2 = "Billy";
            Answer3 = "no one";
            Answer4 = "idk";
            CorrectAnswer = 1;
        };

Removing questions:

To remove a question, it is the same process as adding one. From a }; select it until you get to the question number, then press backspace.


Once you're finished setting up all the questions, publish the game again then go to your order channel in our discord server.


Step 3: Bot Whitelist:

Head to your order channel in our discord server then run -setup. You'll be able to see all of the setup options.


When you're in your channel, run -setup addplace [PLACEID of the app center] this will whitelist your game.
