Rank Center Basic Usage & Set Up

Information:

The Ranking Center is made so that customers can sell ranks in their own group. With the rank center, you're able to easily sell any desired rank and have it automatically set the buyers rank to the specified role.


Step 1: Obtaining Necessary Requirements

In order to configure the rank center, you must first have possession a few items.

These Items include:

  • Valid License
  • Valid Key
  • Personal Order Channel in our discord
  • Fully configured bot (Click here to see how to configure the bot)
  • Download the Template: Click here to download the Template


Once you have requirements above, please continue reading this guide.


Step 2: Configuration

When you have downloaded the template, please open it in studio.

What it should look like when you open it:

Now, publish the game and turn on HTTP Requests, third party teleports, and studio api (optional)


How to publish the game
To publish a game, head on over to the top left corner and hit file.
Then press "Publish to Roblox"

How to turn on required services
To turn on required services, make sure the game is published then head over to the game settings tab.
Now press "Security"
Then turn on allow http requests, third party teleports, and third party sales.


Once you've done that, head over to the Explorer tab and click the little arrow next to ServerScriptService.


Next, double click the module named Configuration and open it.


When you open the module, you should see this. Here is where all of the configuration for the Rank Center is done.

Now, let's talk about what each of these variables do and what the dictionary is used for.


Group Variable :

  • The group Variable is used to define what group this rank center is for. Remove the numbers that are in it and replace it with your group ID. (Click the Click to see arrow below)

Getting group ID + setting variable:

Click to see 1. Head to your group on the ROBLOX page. 2. Go to the top of your search bar and double click on the numbers. 3. Then, right click and press copy. 4. Next, paste that into the Group variable. You're now done with that!


GameKey Variable :

  • The game Key variable is set the place ID. DO NOT CHANGE THIS. You will have to white list the place via bot as seen later.

OneTimeRedemption Variable :

  • This variable if set to true, it will make it so the person who is claiming the rank can only redeem it once. This means that if the person claims a rank, they will not be able to claim that specific rank again.

-- End of Variables--


Ranks Dictionary:


This dictionary contains three values which are vital for the rank center. The Rank ID value, Rank Name value, and the GamePass value. Please see below on how you configure this.

Dictionary Configuration:

Click to see set up
1. Head to your group on the ROBLOX page. 2. Go to the top right, press the three dots, then click "configure group". 3. Press roles. 4. Now, select the role you want to sell and copy its ID. 5. Go back to studio then replace the first RankID variable with that number. 6. Next replace the text in "Rank Name" with the name of the role associated with the ID. 7. Now, go to the gamepass which will give this rank when claimed. 8. Copy the Gamepass ID and paste it into the GamePass variable. Now you're done with the studio set up! Press publish and go to your order channel in our discord. ### To add more ranks: Copy this below and paste it after each }; lua { RankID = 0; -- Change to the Rank ID RankName = ""; -- Put the name of the Rank isFree = false; --<= if you want the rank to be free set it to true and set GamePass to 0 or nil. GamePass = 0; -- set the gamepass ID };
### To remove ranks: I removed three ranks here, if you only want to remove one, do as I did but only select one.


Step 3: Bot Whitelist:

Head to your order channel in our discord server then run -setup. You'll be able to see all of the setup options.


When you're in your channel, run -setup addplace [PLACEID] this will whitelist your game.


Congrats, you've finished setting up the Rank Center. If you encounter any bugs or have questions, please open a support ticket in our discord server and a staff member will gladly assist you.
